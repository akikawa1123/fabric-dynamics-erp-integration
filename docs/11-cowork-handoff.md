# Copilot Coworkへの引き継ぎ

## データ連携

Power AutomateがSharePointへWork Packageを作る。
利用者は調査結果をコピー＆ペーストしない。

## 工場

- 暫定封じ込め会議
- 封じ込め計画書
- 再検査チェックリスト
- アクショントラッカー

## 営業

- 顧客影響確認会議
- 顧客説明PowerPoint
- 初報メール案
- 社内Teams投稿案

## 担当者

Work Packageには解決済みのUPNを含める。
Coworkは参加者の空き時間を調べるが、正式担当者を推測・変更しない。

## 起動

MVPでは利用者がCoworkで次を1回入力する。

```text
未処理の工場品質対応を開始してください。
```

または

```text
未処理の顧客影響対応を開始してください。
```
