# 圧入工程 PFMEA（抜粋）

対象は圧入ステーション ST-07-PRESS。評価指標は S（厳しさ）/ O（発生度）/ D（検出度）/ RPN。

## 故障モード一覧

### 1. 圧入トルク過大（torque_nm > 上限 50.0）
- 影響: 部品・圧入先の損傷、内部応力残留、後工程不良。S=8
- 原因: トルクセンサー校正ドリフト、押込み速度過大、異物噛み込み。O=4
- 現行管理: しきい値監視（warn 48 / fail 50）、defect_flag、Real-Time ダッシュボード監視。D=3
- RPN=96。推奨: 校正周期短縮、異常時のロット隔離（CP-ST07-PRESS 反応計画）。

### 2. 圧入トルク過小（torque_nm < 下限）
- 影響: 圧入不足・抜け、固定不良。S=7
- 原因: 押込みストローク不足、部品寸法ばらつき。O=3
- 現行管理: トルク下限監視、寸法偏差 dimension_dev_um の確認。D=4
- RPN=84。推奨: 治具点検、寸法 SPC 強化。

### 3. トルクセンサー誤差（計測系異常）
- 影響: 真値とのかい離により異常を見逃す/過検出。S=8
- 原因: 校正ドリフト、配線・温度影響。O=4
- 現行管理: 定期校正（INSP-TORQUE-CAL-001）、基準器比較。D=4
- RPN=128。推奨: **校正状態の早期確認を初動に組み込む**（8D-2025-014 の教訓）。

### 4. 寸法偏差過大（dimension_dev_um）
- 影響: 嵌合不良。S=6 / O=3 / D=3 / RPN=54。
- 現行管理: 寸法測定、SPC。
