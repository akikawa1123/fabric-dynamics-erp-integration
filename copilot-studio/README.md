# Copilot Studioオプション

## Connected Agent description

```text
製造品質インシデントについて、Microsoft Fabricの製造・受注・返品データと
品質文書を調査し、工場または営業向けに確認済み事実、原因仮説、
推奨対応、未確認事項を返す専門エージェント。
```

## 追加する内容

- 工場 / 営業の案内
- incident contextの受け渡し
- Foundry Connected Agent
- factory decision Adaptive Card
- Power Automate呼び出し

## 実装条件

Teams利用者の文脈でFabric Data Agentが成功することを先に確認する。
失敗した場合はFoundryのTeams直接公開を維持する。
